# 崔远 · 完整剧情设计

> 清河崔氏旁支庶子，被嫡系发配到落仙城西坊崔记兵库当挂名伙计。在旧库存里翻到一块来历不明的玄铁残片——这是他翻身的唯一筹码。全游戏预算最低的顾客，买装备靠当玉佩。全程靠来铺子买个人装备推进剧情。主角只是卖东西的见证者。

---

## 循环核心

```
来铺子买最便宜的装备 → 用装备进遗迹找另一半碎片 → 结果决定他能否翻身
```

他只来两趟——第一趟倾尽所有买装备，第二趟回来告别。两趟之间，他的命运已经不可逆地写定。

---

## 第一轮：典当购装

**Round 2 · 进店**：一个年轻人在铺子门口站了很久，没进来。他身上的衣服料子是好的——锦边云纹，清河崔氏的织造手艺——但袖口磨白了，领口的绣线松了两道，腰间佩玉黯淡无光。他在看货架上的攻击型法宝。最便宜的那件凡品攻击法宝，标价六十灵石。他看了很久。

然后从袖子里摸出一块玉佩，放在柜台上。正面刻着崔氏族徽，老物件，光泽发暗——不是经常盘的那种。又从怀里摸出一个布包，打开是一块巴掌大的玄铁残片，通体漆黑，断口处隐隐发着暗金色的光。他在崔记兵库的旧木箱底翻到的——账册上没有任何记录。

"掌柜的，这块玉能当多少？"

**小判定 · 典当价格**：崔远身上除了这块玉佩，什么都没有。

- **掌柜给价公道** → 当出八十到一百灵石。够买凡品攻击型法宝加一颗最便宜的轻身丹，还能剩几个铜板。他把铜板小心地收进袖子里，像是收着一份尊严。
- **掌柜压价** → 只当出五十灵石。他站在柜台前默算了好几遍——刚好够攻击法宝，轻身丹买不起了。攥紧了布包，没有还价。不是不想——是不好意思。

无论哪种，他买的都是铺子里最便宜的东西。

> "攻击型法宝——最便宜的就行。再加轻身丹，也最便宜的那种。"

他把布包重新裹好，攥在手里，里头那块玄铁残片硌得他手心发白。走时在门口停了一瞬——不是犹豫，是下了决心。

---

## 第二轮：遗迹结果

**Round 2 · 判定**：崔远攥着铺子里最便宜的装备和一块来历不明的玄铁残片进了遗迹。他是所有顾客里装备最差的——但他手里的残片，可能是所有顾客里最值钱的东西。

- **攻 ≥ 5 且 遁 ≥ 5（买到了入门级装备）**→ 凡品攻击法宝开路、轻身丹甩掉两拨想抢的散修。他摸进遗迹深处一间偏殿——玄铁残片果然有感应。在那里找到了另一块同源碎片。两块碎片靠近的一瞬间，玄铁表面的黑锈自行剥落，露出内里暗金色的器胚纹路。这是一件未完成的上品法宝，不知什么原因被人拆分后分别封印在遗迹内外。

  他没有把这件事告诉任何人——包括西坊崔记兵库的掌柜。当晚，旁支族老们围着那件器胚坐了一整夜。第二天，崔氏旁支派人去落仙城商会重新登记了印信——不再挂靠在崔氏主家名下。

  一个在落仙城当了三年挂名伙计的世家庶子，用一块来历不明的碎片和一件六十灵石的凡品攻击法宝，撬动了崔氏两百年的家业格局。

- **不达标（装备不足）**→ 在遗迹外围碰上了一队散修——五个人，看上了他手里发光的玄铁残片。他身上仅剩的轻身丹甩掉了四个，但最后一个练气九层的散修截住了他。不是打不过——是没有攻击法宝在手，只能空手挡了对方三刀。

  碎片被抢走了。他靠在南坊一间铺子后面的墙上，仰头看着天，看了很久。手上被刀划开的口子已经凝固了，但胸口比伤口更疼——玉佩当了，灵石花了，碎片没了。回到西坊崔记兵库，当票压在枕头底下。

> 崔远只有一个优势——他在崔记兵库三年，摸过上千件法宝，眼力不比任何人差。但眼力不能当饭吃。他缺的不是眼光，是灵石。

# 结局A线：碎片合一 → 衣锦还乡

## 第三轮：告别

**Round 5 · 回铺**：崔远推开了铺子的门。掌柜差点没认出他——换了一身新衣裳，还是锦边云纹，但这次袖口是新的，领口是新的，连腰带的佩玉也换了块成色更好的。不是主家赏的——是旁支族老给他新打的。

身后跟着两个崔氏旁支的账房，抬着一个红漆木箱。木箱打开，里头码着整整齐齐的灵石——五百块。

"掌柜的，这是谢礼。"

**小判定 · 谢礼品质**：

- **R2 装备单属性远超阈值 + 典当价格公道** → 器胚拼合完美无瑕。五百灵石之外，他又从袖子里摸出一份崔氏落仙城分号的进货名录——薄薄一册，写满了崔氏旁支能供货的法宝品类和底价。这不是谢礼，是合作邀约。"掌柜的，以后进货——崔氏分号跟主家一个价，但不抽七成利润。"这意味着铺子的货源从此多了一条路。
- **勉强达标** → 器胚有细微裂纹但不影响品级。五百灵石照给，但不给名录。他说的话简短些："掌柜的，你卖给我的那件凡品攻击法宝，帮我找到了另一块碎片。我们旁支——从现在起不叫旁支了。"
- **达标但典当被压价** → 器胚完美，但谢礼少了一百——四百灵石。他多解释了一句："当初当玉的时候手头紧，这笔我先少给点。等分号站稳了，补上。"不是小气，是算账算惯了的人还没学会大方。

无论哪种，他从袖子里摸出那张万象钱庄的当票——当玉佩的那张，磨得起了毛边。

"这个我已经赎回来了。"

他把当票撕了，碎片随手扔在柜台上，笑了一下。不是张扬的笑——是一个被嫡系压了二十二年的人第一次可以不用低头的笑。

"我要回家了。不是回西坊仓库——回清河。旁支族老们要跟主家摊牌，需要一个见过遗迹的人站在前面。那个人就是我。"

转身推开铺子的门，门外阳光正好。落仙城的石板路上，一个穿着锦边云纹新衣的年轻人昂首阔步地往北去了。

---

# 结局B线：碎片被抢 → 黯然归家

## 第三轮：告别

**Round 5 · 傍晚回铺**：崔远推开了铺子的门。还是那件袖口磨白、领口松线的旧衣裳——不是不想换，是没得换。

他把六十灵石放在柜台上，推给掌柜。

"这个还你。上次买攻击法宝没买到合手的——不是你不卖，是我灵石不够。"

**小判定 · 当票**：

- **典当时价格公道** → 那六十灵石是他攒了一个多月的工钱。当票压在枕头底下，还差二十灵石就能赎回来。他把六十灵石还了铺子——赎玉佩的钱又不够了。但他说这话时语气平静，像是已经习惯了算账永远算不平。
- **典当时被压价** → 勉强凑够了还铺子的六十灵石。当票压在枕头底下，赎玉佩的钱遥遥无期。他甚至没有提这件事——不是忘记了，是说出来也没用。

他把一沓拆开的账本放在柜台上，整整齐齐码好。然后说了句掌柜没料到的话：

"崔氏主家把我召回去了。不是升迁——是旁支有人提议独立，主家要查是谁在背后煽动。我没碎片，没证据，没底气。主家查到我头上，我连说不的能力都没有。"

顿了一下，像是在找一个能让自己接受的说法：

"我起码进过遗迹。不是每个发配到落仙城的人都有这个胆子。"

他转身推开铺子的门。落仙城的夜色里，一个穿着旧衣的年轻人低着头往北去了。那件放在枕头底下的旧当票还压在那里——他不知道什么时候能赎回来。或者永远也赎不回来。

# 三线总览

```
崔远 R2 进店，当玉佩买最便宜的攻击法宝 + 轻身丹
│
├─ 典当价格公道 ──→ 够买攻击法宝+轻身丹
│                       │
│                       └─ R2 遗迹判定
│                              │
│                              ├─ 攻≥5+遁≥5 ──→ 碎片合一
│                              │                     │
│                              │                     └─ R5 衣锦还乡
│                              │                            │
│                              │                            ├─ 属性远超阈值 → 500灵石+进货名录
│                              │                            ├─ 勉强达标 → 500灵石
│                              │                            └─ 达标但典当被压 → 400灵石
│                              │
│                              └─ 不达标 ──→ 碎片被抢
│                                                   │
│                                                   └─ R5 黯然归家
│                                                          │
│                                                          ├─ 典当公道 → 还60灵石，当票待赎
│                                                          └─ 典当被压 → 还60灵石，当票无期
│
└─ 典当价格被压 ──→ 只够买攻击法宝，轻身丹买不起
                       │
                       └─（不达标概率更高，几乎锁定 B 线）
```

## 设计备忘

1. 主角只是卖东西的人。崔远每次来都是顾客——当玉佩、买装备、走人。他不吐露碎片来历，主角也不过问。
2. 所有采购都是个人量级——一件最便宜的攻击型法宝、一颗最便宜的轻身丹。他是全游戏预算最低的顾客。
3. 崔远的线是 A/B 二元结构——没有死亡线。即便装备不达标，他也能活着回来，只是失去了碎片。他的"死亡"不是肉身意义上的——是翻身机会的死亡。
4. 两个小判定——R2 典当价格 + R5 谢礼品质——都不影响大局。R2 典当价格间接影响达标概率（买到两种装备 vs. 只买到一种），但也不是绝对锁死。
5. 崔远的玄铁残片是封印线三块钥匙碎片之一。A 线中他带走了碎片（旁支独立），B 线中碎片流入散修市场——兜了一圈可能回到主角手里。两条线的碎片去向不同，直接影响封印线终局的难度和路径。
6. 崔远是次要种子（◇），剧情线相对精简。他的故事不靠多轮回访——两次进店，一买一别，干净利落。
