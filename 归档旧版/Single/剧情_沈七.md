# 沈七 · 完整剧情设计

> 大炎王朝鸾仪卫铜鳞密探，以南坊茶馆跑堂身份潜伏落仙城。笑嘻嘻的话痨，知道的事情远比一个跑堂应该知道的多。他不是来花钱的——他是来用情报换装备、用闲聊织网的。全程靠来铺子买东西推进剧情。主角只是卖东西的见证者。

---

## 循环核心

```
来铺子买东西 + 聊天摸底 → 用装备深入遗迹执行鸾仪卫任务 → 任务结果 + 带回情报/令牌/人
```

他是唯一一个不进遗迹做属性判定的种子人物——不是不需要装备，是他的战场不在战力上。他靠情报活，铺子是他织网的一个节点。

---

## 第一轮：摸底购装

**Round 5 · 傍晚进店**：茶馆刚歇了灶，沈七就晃进了铺子。进门先不买东西，靠在柜台上跟掌柜聊了一盏茶的工夫——东坊进了什么货、西坊崔家最近账目不对、城南遗迹那边又传出什么新动静。

聊着聊着话锋一转："对了掌柜的，前几天是不是有个穿黑斗篷的来过？那人怪吓人的，你没事吧？"

他问得很随意，但眼神在柜台后面的货架上扫了一圈，像在心里记账。

**小判定 · 摸底结果**：他的问题很多，但从不问"你知不知道"。他问"你见过没有""你觉得呢""后来怎么样了"——每个问题都在试主角的信息量和判断力。

- **铺子货全 + 掌柜对答坦率** → 沈七在铺子里多聊了半盏茶。走时笑得比进门时更真诚了一分——不是套话的笑，是"这人可以打交道"的笑。他在心里给这间铺子贴了个标签：可靠。
- **缺货或掌柜含糊其词** → 他问的问题同样多，但每句"掌柜的真厉害"后面打的折扣也多了一分。走时笑容照旧——但那是茶馆跑堂对谁都一样的笑。

等他把想知道的都摸透了，才"顺便"看看货架。

**本轮购买**：一件趁手的攻击法器 + 一套防御型法宝 + 几瓶应急丹药。付灵石的时候干脆利落——三百到五百灵石，不像散修那样一块一块数。

走之前在门口停了一步："掌柜的，你这铺子位置好——南坊进进出出的人都打你门口过。以后有啥新鲜事，我来找你聊。"

笑着走进暮色里。往茶馆的方向走了几步，然后拐进了一条不在回茶馆路线上的巷子。

---

## 第二轮：遗迹探索（背面）

沈七不像其他种子人物那样在遗迹里做公开的属性检定。他是密探——他的遗迹探索在暗中完成，不需要一个判定回合。

但他买的装备仍然决定了他的任务成果。

**内部判定 · 遗迹成果**：

- **攻 + 防 + 遁综合达标（装备过硬）**→ 在遗迹深处找到了那块传说中能号令上古残军的令牌。同时顺手摸清了遗迹内几处隐藏区域的结构——这些情报对鸾仪卫的价值不亚于令牌本身。秦伯安握着令牌沉默了很久，然后开始修改他对各大仙门的态度——从"谁都不得罪"变成"开始有人站队了"。
- **部分达标**→ 找到了令牌，但遗迹内的陷阱触发了一次，防御法宝碎了一层。令牌是完好的，但他自己多了几道暗伤。回到茶馆后老吴注意到他泡茶时左手有些抖——沈七笑着说"没事，烫了一下"。老吴没追问。跑堂的手被烫不是很正常吗。
- **严重不达标**→ 在遗迹中被迫提前撤离。令牌被苍冥派的人捷足先登——沈七认出对方的功法路数但来不及阻止。回去给秦伯安写了一份长达三千字的检讨。秦伯安看完只说了一句："令牌没了，但你还活着。活着就有下次。"

> 令牌是封印线三块钥匙碎片之一。无论哪种结果，令牌的最终归属都会改变落仙城的势力格局。

---

## 第三轮：命运交汇

**Round 8**：沈七是该来铺子了——但这次来的不只他一个人。或者只有他一个人。取决于一个人的死活。

---

# 结局A线：收编无邪

**触发条件**：殷无邪（P03）在遗迹和追兵决战中存活。

**Round 8 · C002 收编无邪**：铺子门口阳光正好——沈七走了进来，后面跟着一个人。那个人穿着黑斗篷，但这次兜帽没有戴上。

殷无邪站在沈七身后半步的位置，腰间的清心玉佩散发着稳定的青色微光。他看起来和之前不一样了——不是外表，是站姿：以前进门时是贴着墙走的，现在他站在门口正中间。

沈七笑着往柜台上一靠："掌柜的，来给你报个喜。"

他压低了声音，但压不住语气里的得意："血煞宗在落仙城的暗桩——一共七处，名单我已经拿到了。这位——"他朝殷无邪偏了偏头，"——以后不叫逃亡者了。从今天起，他是鸾仪卫的人。"

无邪往前走了一步，从袖中取出那面巴掌大的铜镜，放在柜台上。镜面光滑如水，映出来的不是人的倒影——是几条闪烁着不同颜色光纹的属性。

**小判定 · 铜鳞令品级**：沈七从怀里掏出一块铜鳞令牌。

- **沈七 R5 装备过硬 + 遗迹令牌完整** → 铜鳞令上多了一道银边——不是普通的铜鳞令，是秦伯安特批的"带功铜鳞"。拿着它不止能找沈七一个人——落仙城任何鸾仪卫暗桩都认。他在无邪的事上立了功，令牌含金量跟着涨了。"掌柜的，我们这种人——干的都是不能让人知道的活。但有些事得让人知道。你卖的这些东西，救了不止一条命。"
- **沈七装备达标、令牌完好** → 标准铜鳞令。"以后遇到麻烦——朝廷这边的事，拿这个来找我。"
- **沈七装备勉强、遗迹令牌残缺或丢失** → 铜鳞令照给，但沈七的笑容底下藏着一点紧绷。他不是不满意——他是知道自己这趟差办得不够漂亮，所以令牌的话说得也短："拿着。有用的。"

无论哪种，两人并肩走出铺子。午后的阳光照在南坊的石板路上，无邪的黑斗篷被风吹起一角——底下露出的不再是逃亡者的狼狈，是一身崭新的鸾仪卫深青色短打。沈七拍了拍他的肩："走了，去城主府。秦大人等着呢。"

无邪回头看了铺子一眼，没说话，点了点头。

两个人一起消失在南坊的市声里。

---

# 结局B线：独行采购

**触发条件**：殷无邪（P03）已死在遗迹或追兵决战中。

**Round 8 · AP06E002 独行采购**：沈七推开铺子的门。还是笑嘻嘻的，但笑得比平时安静——茶馆跑堂那种没心没肺的热闹劲少了三分。

"掌柜的，来补点货。"

在货架前转了一圈，买了几件日常用得上的东西——没有上次那么多、那么全。像是在找一个理由在铺子里多待一会儿。

付完灵石，靠在柜台上沉默了一会儿。

**小判定 · 沉默的分量**：

- **沈七装备过硬、令牌在手** → 他完成了自己的任务。但那个本该站在他身后半步的人不在了。他沉默片刻后笑了一下，拍拍柜台："算了。有些人等不到——但我手里有名单了。慢慢清。"语气里是密探特有的冷静——任务没因为少了一个人就停。他走时步伐仍然干脆，但推门前回了下头。
- **沈七装备不足、令牌丢失** → 双重失败。他不仅仅失去了一个能收编进鸾仪卫的人——他连自己的任务都没完成。他没有说令牌的事。只是站了一会儿，然后说了句"掌柜的，有些事急不得"。走出门时步速比平时快——不是赶路，是不想在铺子里多待。

南坊茶馆的灶烟还在冒，老吴还在擦桌子，散修们还在聊着遗迹里的谣言。一切照旧。只是少了一个人。

## 设计备忘

1. 主角只是卖东西的人。沈七来铺子不是找盟友——是来买东西顺便摸底的。主角不需要识破他的身份，不需要配合他演戏。正常卖货就行。
2. 所有采购都是个人量级——一件攻击法器、一套防御法宝、几瓶丹药。没有批量的。
3. 沈七是唯一一个不在遗迹中做公开属性判定的种子人物。他的遗迹探索是"背面叙事"——发生在回合之间，成果取决于 R5 买的装备质量，但不触发单独的事件判定回合。这符合他的身份：密探的工作不该被看到。
4. A/B 线的分叉不取决于沈七自己的属性——取决于殷无邪（P03）的生死。这是唯一一个"自己的结局由别人的属性决定"的设计。沈七是织网的人，网的强度取决于网上的每一个节点。
5. 沈七的令牌是封印线三块钥匙碎片之一。A 线中令牌归属朝廷（秦伯安），B 线中可能归属苍冥派或遗失。令牌去向直接影响封印线终局的势力配置。
6. 沈七是次要种子（◇），但他的线与所有主要种子都有间接交汇——他监控全城。白芷采药时他看见过、江枫进铺子时他打听过、陈洁入城时他注意到、崔远典当玉佩时他在茶馆听人提过、无邪的黑斗篷他更是第一时间就知道了。
7. 铜鳞令品级和遗迹成果的小判定都不影响大方向——沈七始终是鸾仪卫铜鳞密探，始终在落仙城织他的网。区别只在这张网的密度和含金量。
