# 陈洁 · 完整剧情设计

> 龙族族长幼女，化形潜入落仙城，目标是取回遗迹中龙族先祖的「龙魂晶」。压制妖力维持人形期间防御大幅下降。全程靠来铺子买个人装备推进剧情。主角只是卖东西的见证者。

---

## 循环核心

```
来铺子买个人装备 → 用装备去做她自己的事 → 事情的结果 → 下次带着新需求回来
```

每一个剧情节点都对应一次进店购物。她不来闲聊，每次来都买东西。

---

## 第一轮：初探遗迹

**Round 4 · 进店**：买一件防御法宝 + 几颗护体丹。不识货，可能被坑。多给灵石不找零。走时袖口露出鳞片。

**Round 4~5 · 探险**：带着买到的装备独自进遗迹，闯龙骨祭坛三重禁制。

**Round 7 · 结果回铺**：

- **攻≥7 + 防≥7** → 取到龙魂晶。回铺展示晶石，买一瓶回气散路上用。留下万妖岭玉牌——"以后拿着这个来找我"。走时夕阳下影子是龙形。回万妖岭后被任命为和谈副使。

- **攻≥6 + 防≥5** → 空手而归。买几颗疗伤丹回程路上吃。说"破不开第三重"。用矿脉碎片补还上次多给的灵石。回万妖岭后在族会上说"落仙城有个人类商人卖东西没坑龙族"。

- **未达标** → 死去。散修带回焦黑龙鳞来铺子问收不收旧货。万妖岭金色龙鳞落地——宣战。

# 结局A线：龙魂晶 → 使团和谈

## 第二轮：使团入城

陈洁作为副使随使团入城。抽空来铺子买个人用度的消耗品——辟谷丹、回气散、应急疗伤药各几份。谈判期间自用。

买完多说一句："谈判不知道要拖多久，以后缺什么还来你这里。"

**Round 10 · 欢迎宴上的小插曲**（属性判定）：当晚城主设宴接风。席间有修士以"验证龙族实力"为由提出切磋。陈洁服下刚从铺子买的应急疗伤药维持人形稳定。

- **遁速 ≥ 该阶段阈值** → 从容应对，切磋点到为止。全程人形不破，龙族使团的姿态赢得尊重。
- **未达标** → 应对吃力，左袖鳞片失控现出一次。虽然无人追究——使团身份早就公开——但她在宴后独坐许久。

> 不影响和谈，只影响她在人类修士间的个人声望。

## 第三轮：谈判期间的回头采购

谈判拉锯期间，她偶尔会来铺子补丹药。每次都是个人用量。

- R12：买几颗护体丹——谈判桌上天剑宗咄咄逼人，需要维持防御气场
- R14：买一份培元丹——谈判到了关键阶段，状态不能掉
- R16：买几颗回气散——耗了几个月，日常补给见底了

**R12 属性判定 · 谈判桌上的防御**：天剑宗代表在条款上施压，灵压直接逼向妖族使团一侧。护体丹的质量决定她个人承受的压力。

- **防御 ≥ 该阶段阈值** → 面不改色。天剑宗代表收了几分轻视——这个龙族副使不好对付。
- **未达标** → 额头见汗，靠主使解围。和谈继续，但她个人在后续谈判中的话语权略有削弱。

**R14 属性判定 · 长夜谈判**：条约细节连续磋商到下半夜。她服下培元丹维持状态。

- **丹药效果达标** → 抓住草约中一个对妖族不利的条款漏洞，在签字前修正。
- **未达标** → 困乏中漏过该漏洞。次日被天剑宗自行发现并修正——对方补了一句"贵使团审条约不够仔细"。

> 不影响和平结果，但修正权归属会影响使团回万妖岭后的述职评价。

R16 之后：她买完不会立刻走，在铺子里多站了一会儿。龙族不习惯在人类面前久留，这个动作本身就是难得的认可。

> 如果中间回访时铺子缺货——陈洁不会翻脸，但终局对话会变成纯商业口气。因为她的逻辑很简单：能买到好用的东西就常来，买不到就算了。

## 第四轮：和谈成功

和谈签署。当晚她最后一次来。这次不买东西。留下一片龙鳞护符——她自己的鳞片炼制的个人信物。

**护符品质**取决于 A 线全程累计：每次属性判定通过 +1。全部通过 → 本命鳞片，蕴含传承级龙威；部分通过 → 普通龙鳞护符，保命级防御；多数未通过 → 护符黯淡，聊表心意。

> "明天回万妖岭。以后你来——不用玉牌，报我名字就行。"

走时夕阳投下完整龙形影子。

# 结局B线：空手而归 → 鹰鸽僵持

## 第二轮：回程补给

**Round 7 · 回铺**：买几颗疗伤丹 + 一包辟谷丹。够路上吃就行。

> "你卖的东西够好了。是我自己的问题。"

用矿脉碎片补还上次多给的灵石。走时背影单薄。

## 第三轮：私下回访

**Round 9 · 秘密进城**：万妖岭鹰鸽两派僵持，虎族威胁下月投票开战。陈洁以私人身份独自返回落仙城。买一把防身短匕（攻击法宝）+ 几颗护体丹——独自进城，不能让虎族知道龙族族长女儿在战前跑进了人类城池。

买完之后她说了句话：

> "虎族下个月投票。我是族长女儿，我的意见有分量。我在落仙城只认识你一个人类。所以我来问你——你觉得人类值得信任吗？"

她不是在寻求一个战略建议。R4 那件防御法宝在遗迹里帮她活着出来了，这是她判断的参照——装备好不好用，就是她眼中"人类靠不靠谱"的尺度。

她买完东西问完话就走了。主角不需要回答。

**Round 10 · 穿越边境**：买到的装备用于穿越边境返回万妖岭。途中经过虎族领地。

- **攻+防 ≥ 该阶段阈值** → 安全回到万妖岭。用族长信物推迟虎族投票，局部冲突避免。
- **未达标** → 穿越边境时被虎族巡逻队发现，受伤被扣，未能及时赶回。虎族投票通过，局部冲突爆发。

---

# 结局C线：陨落 → 人妖战争

## 第二轮：碎片回铺

**Round 7 · 散修来卖破烂**：一个散修拿着破损的防御法宝碎片和几片焦黑龙鳞来问收不收旧货。碎片背面沾着深蓝色龙血。

同一时间，万妖岭方向金色龙鳞落地——宣战。

## 第三轮：战争阴影

开战消息传开，落仙城进入备战状态。防御类材料涨价。散修们来买装备的频率变高——不是囤货，是真的要上前线。每个人买的量跟平时一样，但来的人多了。

战争持续。散修被征调，铺子的客源从散修逐渐变成军需官。来的还是一个个的人，只是他们的身份变了。

## 对其他人物的连锁影响

| 人物 | 影响 |
|------|------|
| 江枫 | 天剑宗改革派被调往前线，剑道精进路线被军务打断 |
| 白芷 | 各方忙于备战，玄清宗对赵元恪的旧案调查被搁置 |
| 殷无邪 | 战乱中血煞宗趁乱活动，追查名单的窗口期变短 |
| 沈七 | 朝廷情报网转向妖族前线，对城内监控减弱 |
| 崔远 | 崔氏旁支在军需潮中面临机会也面临被吞并的风险 |

## 第四轮：战争终局

战争走向取决于其他种子人物的各自结局。如果其他线推动了足够的和平力量，可能促成多方调停。如果全线崩盘，妖族攻破落仙城，南坊化为焦土。

## 设计备忘

1. 主角只是卖东西的人。陈洁每次来都是顾客，买完东西走人，去做她自己的事。主角不帮她做决定，不接她的订单，不参与她的谋划。
2. 所有采购都是个人量级——几颗丹药、一件法宝。没有批量的。
3. 三条线的分叉来自 R4 的属性判定。B线内部有 R10 的第二次属性判定（穿越边境），决定局部冲突能否被推迟。A线内部有三个不影响大局的小判定（R10欢迎宴遁速、R12谈判防、R14条约审阅），只影响她的个人声望和龙鳞护符的最终品质。
4. 陨落线是唯一无法回头的。陈洁死了就是死了。
5. 所有判定只基于属性数值或前置事件要求，没有模糊的"诚信"概念。B线两次属性判定（R4遗迹 + R10边境）形成因果连锁；A线三次属性判定（R10 + R12 + R14）累积影响护符品质和个人声望。
