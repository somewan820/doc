# 图鉴 UI 需求

## 便利贴形式按钮

- **用途**: 用于图鉴界面中的分类、筛选或功能按钮。
- **视觉风格**: 按钮整体呈便利贴造型，具有轻松、自然、手作感。
- **设计要点**:
  - 提供多种不同颜色版本，用于区分不同入口或分类。
  - 不同颜色按钮之间需要有轻微的样式差异，例如边缘形状、贴纸倾斜角度、阴影或褶皱变化。
  - 整体避免过于整齐统一，需保留自然、随手贴上的感觉。

## 水墨风格按钮

- **用途**: 用于图鉴界面中的特殊按钮或重点操作按钮。
- **视觉风格**: 水墨风格，类似墨水在纸面上横向划过形成的笔触效果。
- **设计要点**:
  - 按钮轮廓应带有墨迹晕染、飞白或边缘不规则效果。
  - 文字区域需要保持可读性，避免墨色过深导致按钮文字不清晰。
  - 整体氛围偏古风、写意，与图鉴界面的书页感保持协调。

## 羊皮纸书页 UI

- **用途**: 作为图鉴内容展示区域或书页背景。
- **视觉风格**: 羊皮纸颜色的书页 UI，带有旧纸张质感和书本立体感。
- **设计要点**:
  - 主体颜色为羊皮纸色，可带轻微纸纹、泛黄、边缘磨损等细节。
  - 左右两侧需要表现出书页厚度，让界面看起来像一本打开的书。
  - 书页边缘可加入层叠页片、阴影或高光，增强立体感。
  - 内容区域需保持干净，方便放置图鉴文字、插图或条目列表。

## 深棕色皮革风树皮

- **用途**: 可作为图鉴装饰边框、标题底板或局部 UI 材质。
- **视觉风格**: 深棕色皮革风，同时带有类似树皮的纹理表现。
- **设计要点**:
  - 色调以深棕色为主，质感厚重。
  - 表面可结合皮革压纹与树皮裂纹，形成自然且有年代感的材质。
  - 需要表现一定厚度，可通过边缘高光、暗部阴影或侧边结构体现。
  - 整体不要过亮，保持沉稳、古朴的视觉感受。

## 回退箭头 UI

- **用途**: 用于图鉴界面的返回、关闭或上一级导航。
- **视觉风格**: 与图鉴整体古风、纸张、皮革或水墨元素协调。
- **设计要点**:
  - 箭头方向明确，交互含义清晰。
  - 可结合手绘、水墨或木质/皮革底座风格。
  - 需要提供常态与按下/悬停状态，方便表现交互反馈。
