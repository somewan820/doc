# 纺市 UI 需求

## 古风木板 UI

- **用途**: 用于纺市界面的标题栏、按钮底板、信息面板或分类入口。
- **视觉风格**: 古风木板 UI，整体偏传统、自然、手工制作感。
- **设计要点**:
  - 需要提供两种不同颜色深度的版本，例如浅色木板与深色木板。
  - 木板表面应有木纹、边缘磨损、钉痕或轻微裂纹等细节。
  - 两种颜色版本应保持同一风格体系，便于在不同层级 UI 中区分主次。
  - 可适当加入阴影或边缘厚度，让木板具有一定立体感。

## 古风背包 UI

- **用途**: 用于纺市中的背包入口、物品栏或资产管理相关界面。
- **视觉风格**: 古风背包造型，适合修仙、古代市集或东方幻想题材。
- **设计要点**:
  - 背包造型应有布料、皮革、绳结或金属扣件等元素。
  - 整体需要有可识别的背包轮廓，便于玩家快速理解功能。
  - 风格需与纺市场景一致，避免现代感过强。
  - 可准备常态与选中/打开状态，用于交互反馈。

## 灵石图标

- **用途**: 用于纺市中的货币、资源、价格或奖励展示。
- **视觉风格**: 东方幻想风格的灵石图标，需具备晶体感和能量感。
- **设计要点**:
  - 灵石形状可以是不规则晶体、打磨宝石或矿石结晶。
  - 颜色可偏青绿、蓝紫或淡金，但需要与整体 UI 色调协调。
  - 图标需要在小尺寸下仍然清晰可辨。
  - 可加入轻微发光、高光或内部纹理，体现“灵力”属性。

## 纺市背景概念图

- **用途**: 作为纺市界面的背景概念参考图。
- **视觉风格**: 古风市集或纺织交易场景，体现东方幻想氛围。
- **设计要点**:
  - 场景需要体现“纺市”主题，可包含布匹、摊位、木架、灯笼、交易区等元素。
  - 背景应服务于 UI 展示，避免主体过于复杂影响前景按钮和信息面板阅读。
  - 色彩氛围可偏温暖、古朴，有市集烟火气。
  - 构图需预留主要 UI 面板、按钮和角色/物品展示区域。
