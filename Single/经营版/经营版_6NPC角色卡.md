# 经营版多 NPC 角色卡

本文件用于经营叙事扩展版剧情。

## 一、总设计原则

- 游戏总长度按约 `50` 回合规划，推荐主流程落在 `R1~R48`，`R49~R50` 作为尾声回合。
- 当前版本包含 `7` 个核心 NPC，全部主要剧情围绕这 7 个人展开。
- 所有数值判定只使用 `attack / defense / speed` 三维属性。
- 商品表中的 `movementSpeed` 在事件判定中对应 `speed`。
- 玩家不直接替 NPC 选命，而是通过卖出的物品改变 NPC 属性值，间接推动他们在关键节点上走向不同人生结局。
- 中后期角色暂时统一设计为 `1 个好结局 + 2 个一般结局`。
- 崔远、费仁这类前期角色剧情更短，暂时设计为 `1 个好结局 + 1 个一般结局`，并在游戏中期前后完成收尾。
- 好结局是剧情主干，需要玩家长期持续提供正确属性，尽量在后段回合触发。
- 一般结局在中段关键节点就可以触发，触发后尽量在 `1` 轮内收尾并退场。
- 关键节点的数值判定不使用“或”逻辑，只使用单一属性门槛或 `&&` 组合门槛。
- 每个结局都必须安排 NPC 本人亲自回店的结局事件，由角色自己报喜或报忧，告诉玩家接下来的动向。
- 结局事件不能用口信、留物、旁人转述等间接方式代替。

---

## 二、整体结构

每个 NPC 的个人线按同一逻辑设计：

1. 先定义固定结局池
2. 再定义关键判定节点
3. 再从好结局主干向外分叉
4. 最后把每条线拆成具体事件节点

建议每个 NPC 使用：

- 中后期角色：`2` 个关键判定节点
- 前期短线角色：`1` 个关键判定节点，主干继续一段后进入好结局

这意味着玩家影响的是：

- 这个人能不能继续往主干走
- 他会不会在中段直接跌入一般结局
- 他能不能一直撑到更后面的回合，拿到真正的好结局

店主在结局事件中的主要身份是倾听者和见证者，不需要替角色解释一切，也不要求角色暴露自己不想说的隐藏目的。

### 二点五、隐式联动说明

- 隐式联动不是 `common` 多人事件。
- 它的实现方式是在角色 `A` 的关键判定节点中，额外加入角色 `B` 的属性值判断。
- 该判断只作用于某个特定 `outcome`，用于触发隐藏结局，不取代角色 `A` 自己的主线判定。
- 当轮文本必须提及另一角色留下的痕迹、影响或出手结果，但不要求角色 `A` 识破对方真实身份。
- 隐式联动必须写清优先级，避免隐藏结局覆盖本应更高优先级触发的主线好结局。

## P21 江枫

### 基本定位

中期核心角色。沉默剑修。真正的问题不是打一场胜负，而是他还能不能重新站回原本那条路。

### 基础属性

| attack | defense | speed |
| --- | --- | --- |
| 5 | 3 | 6 |

### 固定结局

1. 好结局：完成宗门旧事，重归主路  
需一路保持主干推进，直到后段才兑现。

2. 一般结局：中途失手，旧事未了，黯然离城  
在第一次关键节点失败后很快触发。

3. 一般结局：强行出手，事情办成一半却伤了根基  
在第二次关键节点失败后触发。

### 关键判定节点

- 节点 1：R16  
主判定：`attack >= 13`

- 节点 2：R29  
辅判定：`defense >= 8 && speed >= 12`

### 剧情树骨架

```text
江枫主干
R5  AP21E001 初到南坊，买断剑可用的攻击货
└─ R9  AP21E002 再次回店，得知宗门旧事出现窗口
   └─ R16 AP21E003 第一次关键判定：attack >= 13 ?
      ├─ 否
      │  └─ R17 EP21E004 一般结局A：中途失手，旧事未了，离开南坊
      └─ 是
         └─ R22 AP21E005 正面推进，开始真正处理旧事
            └─ R29 AP21E006 第二次关键判定：defense >= 8 且 speed >= 12 ?
               ├─ 否
               │  └─ R30 EP21E007 一般结局B：强行出手，根基受损，只能半途收手
               └─ 是
                  └─ R36 AP21E008 旧事接近收尾
                     └─ R45 EP21E009 好结局：完成宗门旧事，重归主路
```

### 事件节点建议

- R5：第一次进店，只买能继续走的攻击法宝
- R9：第二次来店，第一次明确提到旧事窗口
- R16：第一次关键判定
- R17：一般结局 A 收尾
- R22：主干继续推进
- R29：第二次关键判定
- R30：一般结局 B 收尾
- R36：好结局前置轮
- R45：好结局兑现

## P22 白芷

### 基本定位

中期核心角色。采药炼丹少女。核心不是考核本身，而是她能不能扛住代价，一路走成真正独立的药师。

### 基础属性

| attack | defense | speed |
| --- | --- | --- |
| 5 | 3 | 6 |

### 固定结局

1. 好结局：稳定成长，成为独当一面的药师
2. 一般结局：第一次实作失利，转回低风险慢路
3. 一般结局：硬撑通过前段，却在中段留下严重旧伤
4. 隐藏结局：被无名剑客搭救，避开重伤，却只能落魄回药王谷

### 关键判定节点

- 节点 1：R18  
主判定：`defense >= 9`

- 节点 2：R32  
辅判定：`speed >= 11 && defense >= 12`

### 隐式联动判定

- 联动角色：江枫 P21
- 联动节点：R32
- 联动优先级：
  1. 先判好结局条件 `speed >= 11 && defense >= 12`
  2. 若失败，再判隐藏结局条件 `P21.attack > 50`
  3. 若仍失败，再进入一般结局 B
- 联动表现：
  白芷在当轮 `text` 中会提到一位携带断剑的无名剑客出手相救。她不知道对方是谁，只知道对方出剑极快，替她挡掉了本会留下终身后患的一击。

### 剧情树骨架

```text
白芷主干
R6  AP22E001 来南坊挑药，建立补给关系
└─ R11 AP22E002 采药受伤后回店补货
   └─ R18 AP22E003 第一次关键判定：defense >= 9 ?
      ├─ 否
      │  └─ R19 EP22E004 一般结局A：实作失利，放弃高风险路子
      └─ 是
         └─ R24 AP22E005 继续深入药道，开始承压成长
            └─ R32 AP22E006 第二次关键判定：speed >= 11 且 defense >= 12 ?
               ├─ 是
                  └─ R39 AP22E008 独立承担更高难度事务
                     └─ R46 EP22E009 好结局：成为独当一面的药师
               └─ 否
                  ├─ P21.attack > 50
                  │  └─ R33 EP22E008 隐藏结局：无名剑客搭救，避开重伤，但只能落魄回药王谷
                  └─ 其他
                     └─ R33 EP22E007 一般结局B：人是成了，身上留下难返旧伤
```

### 事件节点建议

- R6：第一次进店，认真挑药
- R11：采药回来，第一次显出受伤痕迹
- R18：第一次关键判定
- R19：一般结局 A 收尾
- R24：主干推进
- R32：第二次关键判定
- R32：若好结局失败，则追加判定江枫 `attack > 50`
- R33：一般结局 B 收尾
- R33：隐藏结局收尾
- R39：好结局前置轮
- R46：好结局兑现

## 五、落地要求

后续如果继续细化到事件 JSON，建议严格按这个顺序往下写：

1. 先写每个 NPC 的固定结局定义
