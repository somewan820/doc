# 经营版 7 NPC 剧情细化稿

本文件用于把当前角色卡中的剧情树骨架，进一步展开成可阅读、可审核、可继续整理为 `events json` 的详细事件稿。

使用原则：

- 本文档按 `R 回合 + 事件编号 + 事件细节` 展开。
- 事件编号沿用当前角色卡规则：
  - 过程事件：`APxxExxx`
  - 结局事件：`EPxxExxx`
- 这里的“事件细节”偏向策划可读稿，不直接等于最终 `json.text` 成品。
- 后续整理成 `events json` 时，可将本稿拆成：
  - 触发条件
  - 本轮提示词
  - 对话目标
  - 玩家可感知信息
  - 结局事件文本
- 所有结局事件都由 NPC 本人亲自回店触发。
- 当前文档已吸收两条隐式联动：
  - 白芷 `R32` 读取江枫 `attack > 50`
  - 陈洁 `R18` 读取殷无邪 `attack > 30 && defense > 30`

## P21 江枫

### AP21E001

- 回合：`R5`
- 性质：主干开场
- 事件细节：

江枫第一次推门进店时，铺子里还没有多少像样的高档货。他站在门口看了一圈，没有立刻开口，只是目光在兵器、符箓、丹药之间迅速扫过，最后停在最能补攻击的一类货上。

他腰间挂着一柄缠着旧布的断剑，旧布包得很紧，像是怕别人看见，又像是怕自己多看一眼。他不谈来历，也不抱怨处境，只问一句：“有没有能让人继续往前走的东西。”

这句“继续往前走”不是漂亮话，而是他当下最真实的需求。他不是来逛店的，也不是来托关系的。他只是需要一件能让自己在下一次出手时不至于太难看的东西。

这一轮里，玩家应感受到：

- 江枫不是没实力，而是正处在一种被压着、不得不临时补短板的状态。
- 他真正缺的是攻击端的正面突破力。
- 他暂时不愿谈宗门旧事，但旧事显然是存在的。

店主在这一轮的作用是：

- 观察出他买攻击货不是为了炫耀，而是为了应付即将到来的事。
- 记住这个人说话很少，但每个字都很实在。

### AP21E002

- 回合：`R9`
- 性质：主干推进
- 事件细节：

江枫第二次来时，比第一次更熟门熟路一些。他不像之前那样站在门口停顿，而是直接走到上次看过的那排货前，先看攻击类，再看护体丹。

他身上有新添的擦伤，不算严重，但显然已经动过一次手。店主如果留意，会发现他断剑上的旧布重新缠过，边角比上次更整齐，像是有人在逼着自己把狼狈收拾干净。

这一次他会比上次多说一点：不是讲经历，而是提到“有个窗口快开了”。所谓窗口，不是好运，而是一件拖了很久的旧事终于出现了能正面处理的机会。他说这句话时没有兴奋，反而很平静，像在说一扇门终于开了一条缝，而他必须决定自己够不够资格进去。

这一轮的重点是把“宗门旧事”从暗示推进到半明示：

- 玩家知道江枫不是在街头争斗，而是在等一个更大的正面处理机会。
- 攻击属性的重要性进一步被确认。
- 江枫开始从“只求凑合”转向“准备认真出手”。

### AP21E003

- 回合：`R16`
- 性质：第一次关键判定
- 条件：`attack >= 13`
- 事件细节：

这一轮不是直接展示结果，而是展示“他决定怎么去面对那扇门”。

如果攻击达标，江枫会在来店时显得异常平静。他不再反复看货，也不再权衡太久，而是只补少量必要消耗，像已经做完决定。他不会说“我要去赢”，只会说“该去把话讲清楚了”。所谓“讲清楚”，对他这样的人来说，往往就意味着剑已经准备好了。

如果攻击不达标，他来店时会明显更沉。他不是不知道自己不够，只是不愿承认。最终他会选择不正面撞上去，而是转去准备更慢、更绕的一条路。这种退让不是胆怯，而是不得不承认自己还没有站上该站的位置。

这一轮玩家应清楚感受到：

- 这是江枫人生真正第一次拐弯的地方。
- 攻击不够，剧情不会只让他“差一点”，而是会直接改变他面对旧事的方式。

### EP21E004

- 回合：`R17`
- 性质：一般结局 A
- 结局类型：中途失手，旧事未了，黯然离城
- 事件细节：

江枫会亲自回店，而且回得很快。

他进门时没有明显重伤，但那种“事情已经过去了”的气息很重。他不会说自己输了，也不会说自己怕了，只会把之前看过、却没买成的一样小东西拿起来看一眼，然后放下。

他会告诉店主，自己不在南坊继续待了。旧事没能了掉，不是因为一点点运气差，而是因为他在真正站上去之前就知道自己不够。他没有抱怨，也没有求安慰，只说“现在再往前撞，只会让事情更难看。”

他不会透露宗门里究竟是谁、发生过什么，但玩家会很清楚：

- 这条线到这里已经断了。
- 江枫还活着，还会继续走，但他这一次没能把人生扳回来。

店主在这一轮更像见证者，只要记住这个人是自己走进来、自己走出去的，没有把不体面留给别人看。

### AP21E005

- 回合：`R22`
- 性质：主干继续推进
- 前置：`AP21E003` 成功
- 事件细节：

这一轮的江枫，已经不是“准备出手”，而是“正在处理后果”。他回来不是为了临时补缺，而是为了应对正面推进后的压力。

他身上的伤比之前多，但人比之前更稳。最关键的是，他已经不再像一个被动流落南坊的人，而像一个重新站回了自己位置的人。只不过，这个位置还没站稳。

他会开始看护体、轻身、回气之类更综合的补给，说明单纯靠攻击冲过去已经不够，接下来考验的是他能不能扛住连带而来的代价。

这一轮的作用是：

- 把主线从“敢不敢出手”推进到“能不能承受出手后的代价”。
- 为 `R29` 的第二次关键判定做情绪和叙事上的铺垫。

### AP21E006

- 回合：`R29`
- 性质：第二次关键判定
- 条件：`defense >= 8 && speed >= 12`
- 事件细节：

这是江枫线里决定“成得体面还是成得残缺”的关键一轮。

如果达标，他回店时虽然疲惫，但整个人是收着的。他不像胜者，也不像败者，更像终于把一件拖得太久的事推进到了能落地的地方。他需要的是最后一轮让自己不垮掉的补给。

如果不达标，他也会回来，但气息会更乱。那种乱不是单纯受伤，而是明显能看出他是硬撑着回来的。他已经把该做的事做了一半，甚至可能已经撬开了局面，但身体、根基或遁走能力没有撑住，结果就是“事情没全坏，但人也没全保住”。

这里应强调：`AP21E006` 写的是江枫在关键一轮里当场如何把局面压住，或者如何在做成一半时把自己赔进去，而不是提前把他后续回店后的去向和心境写完。

这一轮的文本要让玩家感受到：

- 第二次判定不是重复第一次，而是判断一个人有没有资格把胜势真正带回自己身上。
- 不达标的失败，不是“白输”，而是“做成一半，却把自己赔了进去”。

### EP21E007

- 回合：`R30`
- 性质：一般结局 B
- 结局类型：强行出手，事情办成一半却伤了根基
- 事件细节：

江枫会亲自回来，而且回来时状态比前几次都更差。

他不需要别人看出他多狼狈，但狼狈是藏不住的。他会买最基础的疗伤、静气、固本之类的东西，不再挑，也不再在意货值配不配得上自己。

这一次他会承认得更多一点，但仍不会把事情说透。他会告诉店主：“该处理的，算是处理了大半。剩下那点，不是今天能补回来的。”这句话其实已经足够说明结局：他没有完全失败，但也没有完整地赢。

之后他会离开南坊，去一个更适合慢慢养伤、慢慢重整的地方。玩家应明确知道：

- 这不是好结局。
- 但也不是完全沉没。
- 江枫的人生被改写了，只是没能改写得足够漂亮。

这一轮的重点是“事后收束”：上一轮他到底赔掉了什么，如今他怎样接受这个结果，又准备往哪里去，而不是重复描述他在关键一战里如何硬撑着脱身。

### AP21E008

- 回合：`R36`
- 性质：好结局前置
- 前置：`AP21E006` 成功
- 事件细节：

这一轮是好结局前的安静阶段。

江枫回店时，第一次没有那么重的压迫感。他仍然话少，但那种始终绷着的线已经松了一点。他开始买一件“不完全为了打斗”的东西，比如更稳妥的护持物、稍像样的日常用具，或者能让人继续赶路但不再只为拼命的补给。

这说明他的旧事已经接近处理完，只剩最后一点真正落地的收尾。

这里不再重复描述他上一轮是怎么把局面压住的，而是重点表现：经历过那一轮后，他终于开始从“为了旧事活着”转回“准备继续往前过日子”。

这一轮的作用不在高潮，而在让玩家感到：

- 这条长线终于快走完了。
- 他的人生方向已经开始发生根本变化。

### EP21E009

- 回合：`R45`
- 性质：好结局
- 结局类型：完成宗门旧事，重归主路
- 事件细节：

江枫会亲自回店报喜，但不会把“喜”说得很满。

他进门时不会显得张扬，反而比任何时候都更平静。那把断剑还在，只是包着它的旧布终于不是临时缠上去的样子，而像被认真收拾过。他会买一件明显不是为拼命准备的东西，这个动作本身就是最大的变化。

他会告诉店主，事情已经了了，自己接下来不会继续留在南坊。他不必细说宗门里谁输谁赢、谁低头谁让步，只让玩家知道：他终于从一个被旧事拖着走的人，变回了一个能自己决定往哪去的人。

他不会说“谢谢你改变了我的命”，这种台词太直白，也不适合他。但他会让店主清楚地感觉到，这间铺子在他最难的时候，确实给了他能撑到最后的条件。

他走的时候，不狼狈，也不炫耀，只是终于像一个真正上路的人。

## P22 白芷

### AP22E001

- 回合：`R6`
- 性质：主干开场
- 事件细节：

白芷第一次来店时，注意力几乎都在药、材、火候和成色上。她并不把自己当作“需要被照顾的少女”，而是把自己当成一个必须把手上事情做对的人。

她会拿起药瓶闻一闻，或者盯着丹皮的光泽看一会儿，再很克制地指出哪里不够好。语气不重，内容却很准。她不是爱挑刺，而是真的懂。

她来店的直接目标是补一批稳定的药材和护体相关消耗，因为她接下来有采药、试药、实作这些会不断积累小伤的事要做。她眼下最怕的不是一场大败，而是那些不断积起来的小失控。

这一轮玩家应感受到：

- 白芷的线核心是“承受风险的能力”。
- 她不靠攻击翻盘，靠的是稳、忍、准。

---

### AP22E002

- 回合：`R11`
- 性质：主干推进
- 事件细节：

第二次来店时，白芷身上已经出现了采药和实作留下的痕迹。袖口有烧焦的小洞，手上也有没完全褪去的药汁色。她不会诉苦，只会比上次更认真地问一些护体、缓冲反噬、稳住气息的东西。

她开始承认自己不是每一步都能稳稳过掉，也开始承认“保住自己”不是丢人的事，而是继续往前做药的前提。

这一轮是对白芷主线基调的进一步确认：

- 她不是在参加一次单纯的考核，而是在慢慢接近一条高风险成长路径。
- 她的每一步进步，都会伴随身体和精力上的代价。

### AP22E003

- 回合：`R18`
- 性质：第一次关键判定
- 条件：`defense >= 9`
- 事件细节：

这一轮决定她能不能先站稳。

如果达标，她回店时虽然疲惫，但没有乱。她已经扛过了一轮真正可能把人压垮的实作或外出风险，因此有资格继续往更深处走。她不会显得得意，只会变得更笃定。

如果不达标，她也不会一下崩掉，但会明显意识到自己不能继续拿身体硬顶。她会开始接受更低风险、更慢一点的路，而不是继续以为自己能靠硬撑熬过去。

### EP22E004

- 回合：`R19`
- 性质：一般结局 A
- 结局类型：第一次实作失利，转回低风险慢路
- 事件细节：

白芷会亲自回店，而且会很安静地把这件事说明白。

她不会说自己失败了，只会说“接下来的事，不做那么急的了。”这已经足够让玩家明白，她在这次关键节点上没有站住。

她会重新买一些更基础、更保守的东西，说明她接下来的人生还会继续，只是再也不是那条最有冲劲的成长路线。她不是垮了，而是被迫学会先退一步。

这个结局的价值不在惨，而在真实：

- 她没有彻底毁掉。
- 但她也没能往更高处走。

### AP22E005

- 回合：`R24`
- 性质：主干继续推进
- 前置：`AP22E003` 成功
- 事件细节：

这一轮白芷已经开始不只是“补给”，而是在试图建立自己的方法。她来店时会问得更深，不只是“有没有”，而是“什么更稳”“什么会反噬”“什么适合长期做”。

她开始从一个努力扛事的人，慢慢走向一个能判断事情的人。

这一轮的重点是让玩家看到：

- 她正在成长。
- 但真正决定她是不是“独当一面的药师”，还要看下一轮能不能扛住更高层级的压力。

### AP22E006

- 回合：`R32`
- 性质：第二次关键判定
- 主线条件：`speed >= 11 && defense >= 12`
- 隐式联动：
  - 若主线成功失败，则追加判断 `P21.attack > 50`
- 事件细节：

这是白芷线最复杂的一轮，因为这里既决定她能不能迈进真正的好结局，也决定她会不会被另一条人物线以极隐蔽的方式改写命运。

如果主线条件成功，这一轮就应重点写她在当场如何扛住那场最高压的局面：她不是事后才显得成熟，而是在最乱的时候就稳住了手、稳住了气，也稳住了判断，把原本可能一路失控的过程硬生生拉回正轨。这里描述的是“她当时如何撑过去”，而不是她后续回店时已经成长成什么样。

如果主线条件失败，则先不立刻掉入普通坏结果，而是进入隐藏判定：这时如果江枫的攻击力极高，意味着在别处已经成长出一位极强的断剑剑修。他不会正式登场，也不会和白芷相认，但会在她最难的一击到来时，从侧面出手，把本来会留下终身后患的重创挡掉。这里的重点应放在“那一击是如何被挡下来的”“白芷当场经历了什么”，而不是提前把她回店后的去向和心境全部说完。

对白芷本人来说，那只是一个“携带断剑的无名剑客”。
对白玩家来说，这会是清晰的暗示。

如果隐藏判定也不成立，白芷才会落入原本那条“留下难返旧伤”的一般结局。这里同样应把重点放在她当场如何强撑着把事情收住、又是怎样在这一轮里真正受了伤，而把后续“此伤会如何改变她的人生路线”留给结局事件去承担。

### EP22E007

- 回合：`R33`
- 性质：一般结局 B
- 结局类型：人是成了，但身上留下难返旧伤
- 事件细节：

白芷会亲自回来，但这一轮不再重复描述她是如何受伤的，而是重点写她经历那次重伤之后，如今已经把结果想清楚了。

她会告诉店主，事情算是做成了，可自己接下来不会再独自去碰那种高难度事务。她会很认真地问一些以后长期要用的缓养、稳身、止旧伤的东西，也会开始安排自己先回药王谷慢慢把身体养住，再从别的路径把手艺继续磨下去。

这一轮的重点是：

- 她并不是完全失败。
- 但她再也不能像原本那样继续往“独当一面”的高度冲。

她之后的人生仍然有专业、有坚持，只是会一直带着那个代价。

### EP22E008

- 回合：`R33`
- 性质：隐藏结局
- 结局类型：无名剑客搭救，避开重伤，但只能落魄回药王谷
- 前置：
  - `AP22E006` 主线失败
  - `P21.attack > 50`
- 事件细节：

白芷会亲自回店，但这一轮不应再把重点放在“那一剑如何救了她”，而应放在她惊魂稍定之后，对自己处境的重新判断。

她会提到，在最危险的时候，有一个带着断剑的人替她挡下了一击。她并不认识对方，也不知道那人为什么会出手，但这里的叙事功能主要是承接上一轮，而不是再把搭救过程完整重讲一遍。

更重要的是，她会承认自己虽然避开了最坏的身体后果，却也已经明白自己现在还不足以独自承担那种层级的危险。她会决定先离开南坊，回药王谷继续磨练自身，把这次侥幸保下来的路重新走稳，再去找新的方向。

这个隐藏结局的核心是：

- 她没被打成普通失败线里那种“难返旧伤”。
- 但她也没拿到好结局。
- 她的人生被别人的强势成长意外改写了方向。

对白玩家而言，这是两条个人线在暗处发生碰撞。
对白洁本人而言，这只是一次极其不讲理的灾祸。

这里应重点写“陈洁在关键节点当场遭遇了什么”：成功时，是她如何把自己留在遗迹深处；隐藏失败时，是她如何在遗迹里被另一条黑线撞穿；普通失败时，是她如何在最前段就被逼退。回到万妖岭后的局势变化，留给结局事件去写。

## 五、落地要求

后续如果继续细化到事件 JSON，建议严格按这个顺序往下写：

1. 先写每个 NPC 的固定结局定义
2. 再写每个结局对应的触发条件
3. 再写关键节点的属性门槛
4. 最后写事件节点文本

不要反过来先写很多过程事件，再临时决定结局，否则剧情树会重新退化成“过程很热闹，但好结局主干不清晰、一般结局收得不够快”的结构。
